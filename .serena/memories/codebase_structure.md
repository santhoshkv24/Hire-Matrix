# Codebase structure
- Root docs: idea.md, implementation-progress.md, Recruiting System – HireMatrix..txt, progress-gap-analysis.md.
- backend/: Express API service.
  - src/app.js, src/server.js
  - src/config (env, constants, database, mailer)
  - src/controllers, src/routes, src/services
  - src/models (Mongoose schemas)
  - src/middlewares (auth, validation, error)
  - src/utils
  - uploads/resumes for local resume storage
- frontend/: React UI.
  - src/App.jsx, src/main.jsx
  - src/components (AppShell, guards, UI primitives)
  - src/pages (dashboard, jobs, candidates, pipeline, application detail, feedback, decision, exports, notifications, admin users)
  - src/context (AuthContext)
  - src/utils (api helpers)
  - public/ and assets/
