# Project overview
- Purpose: HireMatrix is a recruiting management system for jobs, candidates, applications, pipeline stages, interview feedback, hiring decisions, notifications, dashboards, and CSV exports.
- Backend: Node.js + Express, MongoDB/Mongoose, JWT auth, Multer uploads, Nodemailer email, optional Google GenAI/Vertex AI resume scoring.
- Frontend: React (Vite) with Tailwind CSS, React Router, dnd-kit for kanban drag-and-drop, and framer-motion for UI motion.
- Roles in scope: Admin, Recruiter, Hiring Manager, Interviewer (per current implementation plan).
